# Development Process

## How to start a local environment

You can run a server with our without, a server with a local database or you can just run a local database separatately

### Without a local database

```bash
    yarn dev:parcel
```

### With a local database

```bash
    yarn dev
```

### Just the database

```bash
    yarn dev:db
```
